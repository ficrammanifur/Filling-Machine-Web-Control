Manual Pengguna & Dokumentasi

Memulai

Prasyarat:

  • Board ESP32
  • Relay terhubung ke GPIO 5 (Dingin), 18 (Normal), 19 (Panas)
  • Koneksi WiFi
  • Arduino IDE dengan paket board ESP32

Langkah Cepat:

  1. Upload kode Arduino ke ESP32
  2. Pastikan ESP32 terhubung ke WiFi
  3. Buka antarmuka web ini
  4. Tunggu status "Terhubung"
  5. Mulai kontrol mesin pengisian!

Cara Kerja

Komunikasi MQTT:

Sistem ini menggunakan protokol MQTT untuk komunikasi antara antarmuka web dan ESP32.

Antarmuka Web
→
MQTT Broker
→
ESP32

Alur Pesan:

  • Perintah Dingin: Mengirim "dingin" untuk mengaktifkan relay dingin
  • Perintah Normal: Mengirim "normal" untuk mengaktifkan relay normal
  • Perintah Panas: Mengirim "panas" untuk mengaktifkan relay panas
  • Feedback: ESP32 mengirim status seperti "progress", "done", atau "error" ke topic status

Pemecahan Masalah

Masalah Koneksi:

Problem: Status "Tidak Terhubung" atau "Error"
Solusi:
  • Periksa koneksi internet Anda
  • Segarkan halaman
  • Coba browser lain
  • Nonaktifkan ad blocker atau VPN
Problem: Mesin tidak merespons
Solusi:
  • Pastikan ESP32 menyala dan berjalan
  • Periksa koneksi WiFi pada ESP32
  • Pastikan relay terhubung ke GPIO yang benar
  • Periksa Serial Monitor untuk pesan error
Problem: Perintah tidak berfungsi
Solusi:
  • Tunggu status "Terhubung" sebelum mengirim perintah
  • Periksa apakah ESP32 berlangganan ke topic yang benar
  • Pastikan pengaturan MQTT broker cocok

Spesifikasi Teknis

MQTT Broker: broker.hivemq.com
WebSocket Port: 8884 (WSS)
Topics: filling/perintah, filling/status
Client ID: esp32_filling_relay
Relay Pins: GPIO 5 (Dingin), 18 (Normal), 19 (Panas)
Perintah: "dingin", "normal", "panas"
WiFi Network: Wokwi-GUEST
Baud Rate: 115200

Referensi Kode

Fungsi Utama di Kode ESP32:

wifiConnect(): Menghubungkan ke jaringan WiFi
mqttReconnect(): Menghubungkan ke broker MQTT dan berlangganan ke topic
callback(): Menangani pesan MQTT dan mengontrol relay
processRelay(): Mengaktifkan relay sesuai perintah
client.loop(): Menjaga koneksi MQTT

Konfigurasi Pin:

GPIO 5 Relay Dingin
GPIO 18 Relay Normal
GPIO 19 Relay Panas
